## Computer Vision - Object Tracking with OpenCV and Python
<strong> [oferecido por Coursera Project Network](https://www.coursera.org/projects/computer-vision-object-tracking-opencv-python) </strong>

<p align="justify">Neste curso baseado em um projeto de 2 horas de duração, aprendi como fazer rastreamento de objetos de visão computacional a partir de vídeos. 
No final do projeto, já entendia como funcionam o Optical and Dense Optical Flow, como usar o MeanShift e o CamShist e como fazer um rastreamento único
e um multi-objeto.</p>

- Tarefa 1: Introdução
- Tarefa 2: Fluxo Ótico (Optical Flow)
- Tarefa 3: Fluxo Ótico Denso (Dense Optical Flow)
- Tarefa 4: MeanShift 
- Tarefa 5: CamShift
- Tarefa 6: Rastreamento de objeto único 
- Tarefa 7: Rastreamento de vários objetos 

Pré-requisitos:
Para ter sucesso neste projeto, é necessário ter um conhecimento prévio em Python e OpenCV.
